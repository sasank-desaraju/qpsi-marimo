Biostatistics Module: Hypothesis Testing, Power, and p-values

How to embed in the QPSI Knowledgebase

After deploying to GitHub Pages, each notebook is available at:

https://YOUR-ORG.github.io/qpsi-marimo/notebooks/NOTEBOOK_NAME.html

Embed any notebook with a simple iframe tag:

<iframe
  src="https://YOUR-ORG.github.io/qpsi-marimo/notebooks/hypothesis_power.html"
  title="Hypothesis Testing"
  style="width: 100%; height: 80vh; border: none;"
  loading="lazy"
  allow="cross-origin-isolated"
></iframe>

Available notebooks

  • diagnostic_testing.html — Diagnostic Testing: The 2x2 Matrix
  • likelihood_ratios.html — Likelihood Ratios & Fagan's Nomogram

  • measures_of_association.html — OR vs RR vs NNT
  • roc_curves.html — ROC Curves & Optimal Cutoff
  • screening_bias.html — Lead-Time and Length-Time Bias
  • forest_plots.html — Forest Plots & Heterogeneity
  • correlation_regression.html — Correlation vs Regression

Notes

  • loading="lazy" defers loading until the iframe scrolls into view — important if embedding multiple notebooks on one page.
  • allow="cross-origin-isolated" is needed for SharedArrayBuffer, which the WASM runtime may use.
  • Notebooks are fully interactive — sliders, charts, and all UI elements work inside the iframe.
